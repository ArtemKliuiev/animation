Canvas

HTML5 Canvas – це елемент, який використовується для динамічного відображення графіки та анімації на веб-сторінках. Він дозволяє малювати лінії, квадрати, кола, а також створювати складніші зображення та інтерактивні анімації за допомогою JavaScript. Canvas має розмір, який можна змінювати через атрибути width і height. Всі малюнки виконуються через контекст, зазвичай 2D або WebGL для 3D графіки. Це потужний інструмент для створення візуально насичених веб-додатків, таких як ігри, візуалізація даних та графічні редактори.
